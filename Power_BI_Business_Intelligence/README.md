## Concepts Covered In This Module

**Power Query**
- Connecting to Excel and CSV data sources
- Changing data types
- Checking and cleaning during import
- Applied Steps documentation

**Data Modeling**
- Star Schema design — Fact and Dimension tables
- Creating relationships between tables
- One-to-many relationship configuration
- Understanding filter direction

**DAX — Data Analysis Expressions**
- Measures vs Calculated Columns
- SUM, DISTINCTCOUNT, DIVIDE
- CALCULATE — changing filter context
- TOTALYTD — year-to-date time intelligence
- DATEADD — period shifting
- SAMEPERIODLASTYEAR — year-over-year comparison

**Report Design**
- KPI Card visuals
- Clustered bar charts
- Line charts with legend by year
- Table visuals with conditional formatting
- Slicers — tile and dropdown styles
- Page-level formatting and theme
