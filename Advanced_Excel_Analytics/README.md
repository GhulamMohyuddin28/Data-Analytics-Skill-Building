## Skills Covered In This Module

**Data Cleaning**
- TRIM — removes leading, trailing, and extra spaces
- PROPER / UPPER / LOWER — standardizes text capitalization
- SUBSTITUTE — fixes specific character inconsistencies
- ROUND — normalizes floating point precision
- Go To Special — finds and removes blank rows
- Remove Duplicates — deduplication using unique keys

**Lookups & Logic**
- XLOOKUP — modern cross-sheet dynamic referencing
- INDEX / MATCH — flexible bidirectional lookups
- SUMIF / SUMIFS — conditional aggregation
- COUNTIF / COUNTIFS — conditional counting
- AVERAGEIF — conditional averaging
- Nested IF / IFS — multi-condition logic

**Data Validation**
- Dropdown list validation on categorical columns
- Number range validation
- Custom error messages

**Pivot Tables & Dashboards**
- Pivot Tables with calculated fields
- Pivot Charts connected to slicers
- Interactive dashboard design
- KPI card formatting
