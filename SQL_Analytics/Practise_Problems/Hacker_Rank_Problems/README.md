# HackerRank SQL Practice Problems

Structured SQL practice problems solved on HackerRank,
organized by difficulty and topic. Every file contains
the solution query with a comment explaining the approach.

---

## Problems Solved

Solutions cover these HackerRank SQL categories:

**Basic Select** — Filtering, ordering, basic conditions
**Advanced Select** — CASE WHEN, conditional logic
**Aggregations** — GROUP BY, HAVING, aggregate functions
**Joins** — INNER, LEFT, multi-table queries
**Advanced Joins** — Self joins, complex multi-table
**Alternative Queries** — Subqueries, nested logic

---

## File Naming Convention

Every file is named after the problem it solves.
Example: `city_all_data.sql` solves the problem
asking for all city data from the station table.

---

## How To Read These Files

Each .sql file contains:
1. A comment at the top explaining what the problem asked
2. The SQL query that solves it
3. Where relevant — a comment explaining the approach
