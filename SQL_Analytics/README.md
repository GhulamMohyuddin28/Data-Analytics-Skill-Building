## SQL Concepts Covered In This Module

**Foundations**
- SELECT, FROM, WHERE, ORDER BY, LIMIT
- Aliases, comments, query structure

**Aggregations**
- GROUP BY, HAVING
- SUM, COUNT, AVG, MIN, MAX, ROUND
- COUNT DISTINCT

**Filtering & Logic**
- WHERE with multiple conditions
- CASE WHEN / ELSE
- IS NULL, IS NOT NULL

**Joins**
- INNER JOIN, LEFT JOIN
- Multi-table queries

**Date Functions**
- YEAR(), MONTH(), DATE_FORMAT()
- DATEDIFF() for date arithmetic

**Advanced**
- CTEs — Common Table Expressions
- Window Functions — RANK() OVER (PARTITION BY)
- Subqueries — nested and correlated
- LAG() for period comparison
